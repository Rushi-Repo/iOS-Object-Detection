Installation - 

1. Install latest versions of npm and expo - 

    i)  npm install npm@latest -g   (if you have node installed, update to the latest version)
    ii) npm install --global expo-cli

2. Open a terminal in the project directory and run - 

    i)  npm install  
    ii) expo start

3. If it runs successfully, open http://localhost:19002/. Scan the QR code(bottom left of the dev webpage) with your iPhone scanner app. This should open the app on your phone for testing.
